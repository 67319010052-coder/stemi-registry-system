# STEMI Registry System (Hatyai Hospital)

ระบบจัดเก็บและติดตามตัวชี้วัดคุณภาพการดูแลผู้ป่วยโรคหัวใจขาดเลือด (STEMI) พัฒนาขึ้นเพื่อใช้ในหน่วยงานอายุรกรรมโรคหัวใจ

## คุณสมบัติระบบ (Features)
- **Patient Management:** ลงทะเบียนและจัดการข้อมูลพื้นฐานผู้ป่วย
- **Clinical Data Tracking:** บันทึกข้อมูลอาการ, การวินิจฉัย และปัจจัยเสี่ยง
- **Procedure Logs:** บันทึกข้อมูลการฉีดสีและสวนหัวใจ (Cardiac Cath)
- **KPI Monitoring:** คำนวณเวลา Door-to-Needle และ Door-to-Device อัตโนมัติ
- **Summary Report:** ระบบสรุปรายงานผู้ป่วยรายบุคคล

## เครื่องมือที่ใช้ (Tech Stack)
- PHP (PDO)
- MySQL/MariaDB
- Bootstrap 5
- JavaScript (Data Calculations)
